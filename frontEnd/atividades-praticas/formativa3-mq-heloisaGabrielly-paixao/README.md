# 💻 Simulador do GalaxyTours 🚀
    Este é um projeto desenvolvido para a formativa de Front-End, focado em 
    criar interatividade para um site de uma agência de viagens. Nessa aplicação,
    será possível visualizar mudanças de aparência de acordo com as restrições
    impostas pela Media Query.

## 🛠️ Tecnologias Utilizadas
    1. **HTML5** (Semântica e Acessibilidade)
    2. **CSS3** (Estilização e Feedback Visual)

## 📁 Estrutura do Projeto
    O projeto foi organizado seguindo boas práticas de mercado:
    ├── index.html
    └── src/
        ├── styles/ style.css (Estilos da página)
        ├── scripts/ script.js (Lógica de interação)
        └── images/ foguete.png (Ativos visuais)

## 🎮 Funcionalidades Implementadas
    1. **Media Queries:** Ajuste dos elementos visuais na tela variando entre
    pixels de dispositivos mobile, tablets e desktops.
