# 💻 Simulador do CinePipoca

    Este é um projeto desenvolvido para a avaliação de Front-End, focado em criar interatividade para um site de um cinema. Nessa aplicação, será possível visualizar: mudança de textos, alteração de estilos, contas matemáticas para cálculo de valor, inserção e apagamento de textos e muito mais!

## 🛠️ Tecnologias Utilizadas
    1. HTML5 (Semântica e Acessibilidade)
    2. CSS3 (Estilização e Feedback Visual)
    3. JavaScript (Manipulação de DOM e Eventos)

## 📁 Estrutura do Projeto
    O projeto foi organizado seguindo boas práticas de mercado: / ├── index.html └── src/ ├── styles/ (Estilos da página) ├── scripts/ (Lógica de interação) └── images/ (Ativos visuais)

## 🚀 Funcionalidades Implementadas
    1. Interatividade Visual: Elementos que reagem ao passar do mouse (Hover e manipulação de Classes).
    2. Saudação Dinâmica: Mensagem de boas-vindas que muda conforme o horário local do usuário.
    3. Cálculo em Tempo Real: Formulário que processa e exibe resultados assim que o usuário digita.
    4. Injeção de Conteúdo: Adição de novos itens na tela utilizando manipulação de DOM.
